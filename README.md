# Gunluk Alan

Takvim ve gunluk notlari birlestiren sade, mobil uyumlu bir web uygulamasi.

## Ozellikler

- Takvimden gun secme
- Gune ozel not yazma
- Kucuk yapilacaklar ekleme
- Gorevleri tamamlandi olarak isaretleme
- Secili gunu kopyalama
- Verileri tarayicida saklama
